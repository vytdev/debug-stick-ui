# Debug Stick UI

This add-on is UI a variant of the port of the infamous Debug Stick item
that was exclusive to Minecraft: Java Edition.

[**Skip to the download page**][curseforge-dl]

Found the [Debug Stick][debug-stick] add-on a bit hard to use? Try this one.

This add-on is an official fork of the Debug Stick port to Bedrock Edition.

![overview][img-overview]

## 🎁 How To Get It?

1. [Download the add-on][curseforge-dl] for your Minecraft version.
2. Import the add-on to Minecraft.
3. Activate it in your world.
4. Open your world!
5. Type `/give @s vyt:debug_stick_ui`, or find it in the Creative menu.
6. That's it!

## 📂 Where To Download?

To get the **correct version for your Minecraft**, please use our CurseForge
download page. It can easily help you sort and find the version that works
with your game.

You can also download directly here on MCPEDL, though it may be a bit
harder to ensure you're selecting the right version.

**Important:** Only download from CurseForge, the official MCPEDL page, or
our official GitHub repo. Other sites may host *unofficial, outdated, or
unsafe copies* that could contain malware or break your game.

### 📦 On [CurseForge][curseforge-dl]

If you're on mobile, switch to "Desktop Mode" so the download button
appears. Then:

1. Click **"Filter By"** (on the left)
2. Select your Minecraft version
3. Pick the matching add-on release
4. Click **"Download"** (the middle button, not the one on the far right)

## 📎 Version Naming

We've matched our add-on version numbers with Minecraft Bedrock since 26.0.

- **1st &amp; 2nd number**: The Minecraft version the add-on works with. If
  your game is older, it might not work.
- **3rd number**: The add-on's patch number. It doesn't have to match your
  Minecraft version &mdash; only the first two numbers matter.

This new version system makes it easier to find the correct version.

*We don't strictly follow [SemVer](https://semver.org).*
